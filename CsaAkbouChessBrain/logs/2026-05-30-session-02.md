## Corrections de bugs

### Sécurité — route admin dans robots.txt
- **Commit `d99c6a6`** — `robots.txt` + `public/robots.txt`
- `Disallow: /gestion-csa-2025` exposait la route admin secrète publiquement. Supprimé.

### React — duplicate key dans Footer
- **Commit `d93f66c`** — `src/components/Footer.tsx`
- `key={item.text}` sur les cards Contact produisait des clés `""` identiques au chargement (avant que site_config réponde). Remplacé par `key={item.id}` stable.

### Performance — lag sur la page About
- **Commit `3f96933`** — `src/index.css`, `src/hooks/useScrollReveal.ts`, `src/pages/About.tsx`
- `filter: blur(3px)` supprimé de tous les keyframes `reveal-*` (coûteux GPU).
- `useScrollReveal` : les éléments déjà dans le viewport au montage passent directement à `isVisible=true` sans flash `opacity-0`.

### Overflow mobile — page About
- **Commit `3f96933`** — `src/pages/About.tsx`
- Section Story : `overflow-hidden` ajouté pour contenir les éléments absolus (`-inset-4` glow, `-right-5` badge) qui débordaient du viewport.
- Section Members : `overflow-hidden` ajouté pour contenir les cards mobiles dont le grid auto-dimensionnait sa colonne au-delà du container.
- Wrapper table desktop : `overflow-hidden` → `overflow-x-auto` pour permettre le scroll horizontal sur tablette.
